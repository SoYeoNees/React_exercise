1. 제이쿼리 링크

https://jquery.com/ (제이쿼리 가서 다운받아서 사용) Google CDN

https://developers.google.com/speed/libraries?hl=ko#jquery

버전은 3.7버전까지 나왔지만 1.12.4버전 사용

2. 제이쿼리의 순서

Google CDN script 이전에 제이쿼리를 사용하면 작동이 안된다.

Google CDN script 와 HTML 요소가 읽힌 후에 작동해야 하기 때무에 일반적으로 body아래에 작성한다.

3. 콘솔창에 제이쿼리 입력해보자!

$

첫번째 div에 fadeOut을 적용해보세요

$($('div')[0]).fadeOut();

4. ID요소 선택

$('#idH3')

ID가 idH3인 요소에 fadeOut을 적용해보세요.

$('#idH3').fadeOut();

5-1. CLASS요소 선택

5-2. CLASS요소 선택

$('.clsH3')

첫번째 .clsH3요소만 fadeOut을 적용해보세요.

$($('.clsH3')[0]).fadeOut();

6. CLICK 이벤트

$('.clickH3').click(function () { $('.clickH3').fadeOut(); });

7. css 변경

$('.cssH3').css('color', 'red');

8. Text 변경

$('.textH3').text('TEXT');

string으로 넣지 않으면 에러가 생겨요.
